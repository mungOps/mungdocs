    Linux란 #


    - Linux는 1991년 Linus Torvals가 개발한 운영체제의 하나이다. - Linux는 Unix 기반으로 만들어진 운영체제로 Unix가 애초부터 통신 네트워크를 지향해 설계된 것 처럼 Linux 또한 서버로 작동하는데 최적화 되어있다. - Linux는 자유 소프트 라이센스로 누구나 소스코드를 활용, 수정 및 재 배포가 가능하며, 이를 기반으로 지속적인 업그레이드가 이루어진다.
