    ERP #


    • ERP란 Enterprise Resource Planning (전사적 자원관리)를 의미한다.
    • 주로 기업내의 모든 업무 기능들이 조화롭게 발휘할 수 있도록 지원하는 어플리케이션들의 총 집합으로 차세대 업무시스템 (통합 정보시스템)을 의미한다.
    • ERP의 출발은 초기 MRP (Mastered Requirement Plainning)에서 진화되었으며, 점차 업무 기능들이 늘어나며 현재도 끊임없이 IT기술과 함께 발전하는 분야이다.
