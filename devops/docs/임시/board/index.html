    Main Board #

     

    • main board란 서버의 주요 구성 부품(CPU, Ram, GPU, Nic 등)들을 설치, 연결, 조절하는 주 회로 기판(main circuit board)을 메인보드(main board)라고 한다.

    • 각 부품들이 하나로 연결되어 온전한 기능을 할 수 있게 해주는 회로가 존재하는 데, 이 회로들을 통해서 전기를 각 부품에게 배분해주는 역할을 하는 부품이 메인보드이다.

    • main board가 없으면 cpu가 램으로 처리된 정보를 보낼 수 없다.

    • 메인보드의 ROM에는 bios라는 일종의 작은 운영체제가 들어있다.

    • 연결된 모든 장치는 메인보드의 bus를 통하여 정보를 교류하며 그 흐름을 메인보드가 관리한다.

    • 전원공급장치(PSU)에서 공급된 전기는 메인보드에서 다시 전압을 조절하여 프로세서와 메인보드 및 각종 전자 장치에 공급됨

    • 메인보드에는 ram을 꽂을 수 있는 ram slot과 cpu slot, 내장 사운드 카드와 내장 랜카드, 외장 그래픽 카드, 사운드 카드, tv 수신 카드 등을 꽂을 수 있는 pci 슬롯을 제공함

    • 즉, main board란 각 파트의 브릿지 역할을 하는 토대이다.

    #

     

    Main board Part #

    #

    • PSU (Power Supply Unit) : 전원공급장치

    • NIC (Network Interface Controller) : 외장 네트워크 인터페이스 카드 / IPMI는 내장

    • PCI (Peripheral Component Interconnect Bus) : GPU, M.2 등 다른 파트를 장착하기 위한 slot

    • CPU : 중앙 처리장치

    • Memory : 메인 메모리로 주로 RAM(Random Access Memory)를 의미

    • R/C (Raid Controller) : R/C는 각 Disk의 raid 정보를 가지고 있으며, 이를 관리하는 파트

    • FAN : 전방에서 후방으로 공기를 순환시키는 선풍기

    • Disk : 비휘발성 메모리로 데이터를 저장

    • Case : 내부의 파트를 보호하기 위한 Case

    #

     

    • Memory slot은 하기 사진과 같이 채널을 이루고 있다.
