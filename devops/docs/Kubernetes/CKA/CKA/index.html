    CKA #

    #

    #

    • CKA 자격증은 Certified Kubernetes Administrator를 의미하며, Kubernetes 플랫폼에 대한 숙련도를 Linux Foundation에서 검증해주는 자격증을 의미

    • CKA 자격시험은 실기형 시험으로 Kubernetes 환경에서 명령어를 활용해 리소스를 다루는 형태로 문제가 출제되며. 2시간의 시험 시간 동안 66/100점 이상이면 시험에 통과
