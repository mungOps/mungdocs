    #

    1. Deploy a pod named nginx-pod using the nginx:alpine image #


    Answer 
    $ k run nginx-pod --image=nginx:alpine -o yaml > 1-1.yaml
$ k apply -f 1-1.yaml

$ k get pods
NAME            READY   STATUS    RESTARTS   AGE
pod/nginx-pod   1/1     Running   0          59s


    2. Deploy a messaging pod using the redis:alpine image with the labels set to tier=msg #


    Answer 
    $ k run messaging --image=redis:alpine --labl=tier=msg -o yaml > 1-2.yaml
$ k apply -f 1-2.yaml

$ k get pods -L tier
NAME        READY   STATUS    RESTARTS   AGE     TIER
messaging   1/1     Running   0          2m46s   msg


    3. Create a namespace named apx-x9984574 #


    Answer 
    $ k create namespace apx-x9984574

$ k get ns
NAME              STATUS   AGE
apx-x9984574      Active   1s


    4. Get th list of nodes in JSON formaget and store it in a file at /opt/outputs/nodes-z3444kd9.json #


    Answer 
    $ k get nodes -o json > /opt/outputs/nodes-z3444kd9.json
